2 矩阵

2.1 矩阵的基本概念

由m$\times$n个数aij(i=1,2,…,m;j=1,2,…,n)按一定顺序排成的m行n列的矩阵数表，称为m$\times$n矩阵，简称矩阵。矩阵用黑体的大写英文字母表示，记作：

2.2 几种特殊矩阵

  1. 零矩阵：元素均为0的矩阵
  2. 三角矩阵：主对角线上方元素均为0的n阶方阵称为n阶下三角矩阵，主对角线下方元素均为0的矩阵称为n阶上三角矩阵。上三角矩阵与下三角矩阵统称为三角矩阵
  3. 对角矩阵：主对角线之外的元素均为0，而主对角线元素不全为0的n阶方阵
  4. 单位矩阵：主对角线元素均为1的n阶对角矩阵
  5. 数量矩阵：主对角线元素均为a的n阶对角矩阵
  6. 梯形矩阵：设A=(aij)m$\times$n,若当i＞j时，恒有aij=0,且各行第一个非零元素前面零元素的个数随行数增大而增多，则称该矩阵为上梯形矩阵
  7. 转置矩阵：把矩阵A的行换成同序数的列而得到的新矩阵，记作AT
  8. 对称矩阵：设A为n阶方阵，如果满足AT=A,则称A为n阶对称矩阵
  9. 反对称矩阵：设A为n阶方阵，如果满足AT=-A,则称A为n阶反对称矩阵
  10. 分块矩阵：设A=(aij)m$\times$n，将矩阵A用若干条纵线和横线分成许多小矩阵，每个小矩阵称为A的一个子块

2.2 矩阵的运算

交换律：A+B=B+A;

结合律：(A+B)+C = A+(B+C);

将数$\lambda$与矩阵A=(aij)m$\times$n的乘积记作$\lambda$A，规定$\lambda$A=($\lambda$aij)m$\times$n

$\lambda$A = A$\lambda$

($\lambda$$\mu$) A= $\lambda$($\mu$A)

($\lambda$+$\mu$)A = $\lambda$A + $\mu$A

$\lambda$(A + B) = $\lambda$A + $\lambda$B

注意：

  1. 矩阵乘法不满足交换律，这是因为AB与BA不一定有意义。即使AB与BA都有意义，也不一定满足AB=BA
  2. 在矩阵乘法中，“若AB=0，则A=0或B=0”这个结论不一定成立
  3. 矩阵乘法的消去律不成立，即“若AD=AC，则D=C”这个结论不一定成立

矩阵的转置：

  1. (AT)T = A
  2. (A + B)T = AT + BT
  3. (KA)T = KAT
  4. (AB)T = BTAT

方阵的行列式
设A，B是n阶方阵，k属于R

  1. |AT| = |A|
  2. |kA| = kn|A|
  3. |AB| = |A||B|
  4. |AB| = |BA|

设A为n阶方阵，若|A|≠0，则称A为非奇异矩阵，否则称为奇异矩阵

伴随矩阵：

  1. AA* = A*A = |A|E
  2. |A*| = |A|n-1

2.3 初等变换

  1. 对调两行（列）
  2. 以数k≠0乘以某一行（列）中的所有元素
  3. 把某一行（列）的所有元素的k倍加到另一行（列）对应元素上去

若矩阵A经过有限次初等行变化变成矩阵B，则称矩阵A与B等价

设A是m$\times$n矩阵

  1. 矩阵A总可以经过若干次初等行变换化为行梯形矩阵
  2. 矩阵A总可以经过若干次初等行变换化为行最简形矩阵

初等矩阵：

  1. 把单位矩阵中的第i，j两行互换（或第i，j两列互换）得到第一种初等矩阵E(i,j)
  2. 把数k≠0乘以单位矩阵的第i行（或第i列），得到第2种初等矩阵E[i(k)]
  3. 把数k乘以单位矩阵的第j行加到第i行上（或把数k乘单位矩阵的第j列加到第i列上），得到第3种初等矩阵E(i+j(k),j)(或E[j,i+j(k)])

左行右列：对矩阵A左乘相当于对A实施同种行变换，右乘相当于对A实施列变换

2.4 逆矩阵

如果方阵A可逆，则A的逆阵是唯一的

方阵A可逆的充要条件是A-1 = $1 \over |A|$A*

设A为n阶方阵

  1. 若A可逆，则A-1也可逆，且有(A-1)-1 = A
  2. 若A可逆，则AT也可逆，且有(AT)-1 = (A-1)T
  3. 若A可逆，则kA也可逆，且有(kA)-1=$1 \over k$A-1(k≠0)
  4. 若A和B均为同阶可逆方阵，则AB，BA均可逆，且有(AB)-1 = B-1A-1
  5. 若方阵A可逆，矩阵B，C满足AB=AC或BA=BC，则有B=C

解矩阵方程：

  1. 矩阵方程为AX=C，其中矩阵A可逆
    1. 等式两边左乘A-1,得X=A-1C
    2. 使用初等变换解矩阵方程，即(A|C)---->(E|X)
  2. 矩阵方程为XB=C，其中矩阵B可逆
    1. 等式两边右乘B-1，得X=CB-1
    2. 使用初等变换解矩阵方程，即
      ($B \over C$) = ($E \over X$)

2.5 矩阵的秩

设A为m$\times$n矩阵，矩阵的秩有下列性质

  1. 0 ≤ r(A) ≤ min{m,n}
  2. 由于转置行列式的值不变，所以r(AT)=r(A)
  3. 可逆矩阵为满秩矩阵，不可逆矩阵为降秩矩阵
  4. 设A为m$\times$n矩阵，P,Q分别为m，n阶满秩矩阵，则r(A)=r(PA)=r(AQ)=r(PAQ)

设有矩阵A与矩阵B

  1. 若矩阵A与B等价，则r(A) = r(B)
  2. 若A为m$\times$n矩阵，B为m$\times$s矩阵，则max{r(A),r(B)}≤r(A,B)≤r(A)+r(B)
  3. 若A，B均为m$\times$n矩阵，则r(A±B)≤r(A)+r(B)
  4. 若A为m$\times$n矩阵，B为n$\times$s矩阵，则r(A)+r(B)-n≤r(AB)≤min{r(A),r(B)}
  5. 设A为m$\times$n矩阵，B为n$\times$s矩阵，若AB=0,则r(A)+r(B)≤n
