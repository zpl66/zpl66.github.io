附录

1. 公式

1.1 万能t

词根结尾+t含义不变，如：

dic = dict

spec = spect

doc = doct

且一般不会予以翻译，比如bullet翻译成芭蕾，jacket翻译成夹克

1.2 屌丝定律

当一个词根的尾辅音是d的时候，那这个词根就一定由尾辅音是s的情况。如：

cad = cas

ced = ces

cid = cis

1.3 n+元音

n+任何元音都表示否定的意思，na,ne,ni,no,nu

2. 通假

  1. v和w通假
  2. n和m通假
  3. e和i通假

3. 后缀

  • -eo:xx东西，xx的人，形容词后缀
